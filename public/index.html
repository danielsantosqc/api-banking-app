GET :id

Ao realizar un method GET com envio de um parametro id:, a respososta sera un json(): con os dados de esse usuario

Exemplo:

http://host/users/f5931c36-71f2-4d59-833a-74467a915a20



            {
              "id": "f5931c36-71f2-4d59-833a-74467a915a20",
              "name": "Norman Rohan",
              "age": 78,
              "email": "NormanRohan90@hotmail.com",
              "phone": "+55 85 978942650"
            }

PATCH :id

Para substituir um dado parcial do usuario

Ao realizar un method PATCH devera enviar um parametro id:, a respososta sera un json(): con os dados de esse usuario ja atualizado

Exemplo:

http://host/users/f5931c36-71f2-4d59-833a-74467a915a20


          //Deve-se respeitar o tipo  de formato do/dos campos a serem substituidos,
          {
            "name":  "Norman Rohan",              //String
          }

POST

Ao realizar um method POST, devera enviar os dados em formato json no corpo do requuest (body), no formato especificado abaixo. Dedve se respeitar o tipo de dato de cada (name: value)

Exemplo:

http://host/users/


          //O nome e valor (name: value) que nao sejan enviados serao prenchidos em branco (undifined)
          {
            "name": "Norman Rohan",              //String
            "age": 78,                           //number
            "email": "NormanRohan90@hotmail.com", //String
            "phone": "+55 85 978942650"           //String
          }

DELETE :id

Apagar os dados de um usuario

Ao realizar un method DELETE devera enviar um parametro id:, a respososta sera un json(): confirmando que o usuario foi removido

Exemplo:

http://host/users/f5931c36-71f2-4d59-833a-74467a915a20


          //sera respondido com a siguinte mensagem,
          {
            "message":  "Usuario removido",              //String
          }
